# 课程介绍

- 理论和实践相结合
- 将编程、开发知识融会贯通
- 熟悉移动互联网开发流程
- 熟练使用工具进行开发、测试、部署、维护

# 课程安排

- 5周，每周9节课
- 周一1-5，周二6-9
- 实践为主，授课为辅

# 考核形式

2-3人一个小组，开发一个完整的**移动互联网**项目

- 课堂表现10%
- 平时汇报10%
- 团队合作、项目开发活跃度10%
- 期末汇报10%
- 文档质量、完整性30%
- 项目完成度10%、创新性10%、工作量10%
