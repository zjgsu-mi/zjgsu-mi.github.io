# 产品原型

- 产品设计方案的表达
- 产品设计界面的展示
- 功能与交互的示意
- 与其他人员沟通的依据

# 为什么做原型设计?

- 便于产品经理对产品的整体把握
- 能使产品需求评审更高效
- 进行产品的视觉设计和功能开发

# 需要考虑的问题

- 此原型的用户是谁？
- 原型需要达到的目的是什么？
- 用户的使用场景是什么？
- 需要提供什么样的用户体验？
- 后续的工作流程如何把握？

# 基本步骤

1. 分析产品功能（思维导图）
2. 根据产品功能，构建页面目录
3. 根据页面目录进行原型图绘制
